Question

Write a program that takes a list of strings and prints them, one per line, in a rectangular frame of asterisks.
The width of the rectangular frame includes a space on either side of the longest word and the asterisk border.
For example the sentence "Hello World in a frame" gets printed as:

*********
* Hello *
* World *
* in    *
* a     *
* frame *
*********

The program is expected to take using input using 'raw_input' and to 'print' the output.
