Question 3

Write a program that takes an input string that describes a calculation (eg: “1 + 3 - 1 =”) and calculate the answer to the calculation.
If the string does not end with an “=”, it is considered to be invalid.
Strings will only contain + and - so you do not need to account for arithmetic precedence.

For example:
“1 + 5 -2 =”	-> 4
“1 + 3”		-> invalid input

The program is expected to take using input using 'raw_input' and to 'print' the output.
